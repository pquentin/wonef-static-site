WoNeF : WordNet du Français

WoNeF

WoNeF est une traduction automatique de WordNet réalisée à partir de dictionnaires bilingues et d'espaces sémantiques. Chacun des synsets de WordNet est rempli par des littéraux (noms, verbes, adjectifs, adverbes) français.

Versions

Trois versions différentes sont disponibles au téléchargement : une version à haute précision, une version à haut F-score et une version à haute couverture.

Évaluations

Les différentes versions ont été évaluées avec rigueur, la version la plus précise atteignant 93% de précision. La vérité-terrain ayant servi à l'évaluation est librement disponible.

Accessible

WoNeF est librement disponible sur Internet, mais il est aussi possible de le télécharger sous format XML. Une API Python rend son intégration très facile.
