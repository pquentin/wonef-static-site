WoNeF

WoNeF · WordNet du Français

WoNeF is an automatic WordNet 3 translation created from bilingual dictionaries and a web-based syntactic language model. Every WordNet synset is filled with french literals (nouns, verbs, adjectives, adverbs). WoNeF is free as in freedom.

Try WoNeF

Adapted

Three different versions are available : a high-precision version, a high F-score version and a high-coverage version.

Free

WoNeF is under the CC-BY SA 3.0 FR license, which allows using WoNeF as you wish. It's also an open project where anyone will be able to add new translations and correct existing ones.

Evaluated

The three versions have been evaluated rigorously, the most precise version reaching 93% precision. The gold-standard used for evaluation is freely available.

Accessible

You can browse WoNeF on this website, but you can also download it as an XML file. The standard DEBVisDIC format maximizes its usefulness.
